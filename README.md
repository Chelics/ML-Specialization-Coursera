
# 吴恩达机器学习专项 (ML Specialization) - 个人练习仓库

本项目是我学习吴恩达教授 (Andrew Ng) 在 Coursera 上的 **《Machine Learning Specialization》** 专项课程的个人代码仓库。

特别感谢 Andrew Ng 和 Coursera 团队创建了这门浅显易懂的机器学习入门专项。

本仓库 Fork 自 [karkir0003/ML-Specialization-Coursera](https://github.com/karkir0003/ML-Specialization-Coursera) 提供的空白作业模板，用于存放我自己的代码实现、笔记和实验。

---

## 课程结构

这个专项课程分为三门子课程：

1. **C1: Supervised Machine Learning: Regression and Classification** (监督学习：回归与分类)
2. **C2: Advanced Learning Algorithms** (高级学习算法)
3. **C3: Unsupervised Learning, Recommenders, Reinforcement Learning** (无监督学习、推荐系统与强化学习)
